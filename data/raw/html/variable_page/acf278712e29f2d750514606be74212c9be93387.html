Mental Health Service Provider Classification Indicator — Ever in Calendar Year

This variable is contained in the following files:
T-MSIS Analytic File Annual Provider
SAS Name
MH_SRVC_PRVDR_CLSFCTN_IND

Indicates whether any of the provider’s classification codes values map to the mental health service provider category; ever in the calendar year.

Comments

This variable = 1 when the description associated with the provider ID number explicitly references a mental health service provider (e.g., Addiction Psychiatry, Geriatric Psychiatry, Adult Mental Health, Psychiatric Unit, etc.) based on the following assignment:

PRVDR_CLSFCTN_SYS = ‘1’ and taxonomy value is ('101Y00000X', '101YM0800X', '101YP1600X', '101YP2500X','101YS0200X','102L00000X', '102X00000X','103G00000X', '103GC0700X', '103K00000X','103T00000C','103TA0700X', '103TC0700X', ’103TC2200X','103TB0200X', '103TC1900X', '103TE1000X','103TE1100X', '103TF0000X', '103TF0200X', '103TP2701X', '103TH0004X',' 103TH0100X', '103TM1700X', '103TM1800X', '103TP0016X', '103TP0814X', '103TP2700X', '103TR0400X', '103TS0200X', '103TW0100X', '104100000X',' 1041C0700X', '1041S0200X', '106E00000X', '106H00000X', '106S00000X', '163WP0808X','163WP0809X', '163WP0807X', '167G00000X', '1835P1300X', '2080P0006X', '2080P0008X','2084B0040X', '2084P0804X','2084F0202X','2084P0805X', '2084P0005X', '2084P0015X', '2084P0800X', ’225XM0800X', '251S00000X', '252Y00000X', ’261QM0801X', '261QM0850X', '261QM0855X', '273R00000X','283Q00000X','3104A0630X', '3104A0625X', '310500000X', '311500000X', '315P00000X','320600000X','320800000X', '320900000X', '322D00000X', '323P00000X', '363LP0808X','364SP0807X', '364SP0808X', '364SP0809X', '364SP0810X', '364SP0811X', '364SP0812X', '364SP0813X', '385HR2055X', '385HR2060X'), OR

PRVDR_CLSFCTN_SYS = ‘2’ and provider specialty is '26','27','62','68','86', OR

PRVDR_CLSFCTN_SYS = ‘3’ and provider type is '26', OR

PRVDR_CLSFCTN_SYS = ‘4’ and authorized category of service is '044','045','046','048','049'

All other values = 0
