Claim Line Count - Original

This variable is contained in the following files:
T-MSIS Analytic File Pharmacy T-MSIS Analytic File Other Services T-MSIS Analytic File Long Term Care T-MSIS Analytic File Inpatient
SAS Name
CLL_CNT

The total number of lines on the claim as recorded by the state when TMSIS data submitted

Comments

The value is what the provider submitted on the claim. There can be inaccuracies. See also CLL_CNT_CALC.

SOURCE: T-MSIS Analytic File (TAF) Claims

1 – XXX
Equals the count of the claim lines submitted on the original claim.
Code Code value
Null/missing source value is missing or unknown
