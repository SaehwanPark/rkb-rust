Dialysis service type code 1

This variable is contained in the following files:
Medicare Provider Analysis and Review
SAS Name
ESRD_SETG_IND_1_CD

The 1st code indicating the type of dialysis received by the beneficiary during the stay. Up to five (5) of these 2-position codes may be present.

Comments

This field is derived from the presence of the dialysis revenue center codes (080X, 082X, 083X, 084X, 085X, and 088X) listed below on any of the claim records included in the stay.

This variable appears 5 times, where in the variable name is a value 1:5.
