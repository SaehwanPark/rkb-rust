1 For all columns under "Year of Data File":  X = variable is present; '-' indicates not present, or not populated; LC = length change for variable; VC = value change.
2 Prior to 2015, CCW was required to encrypt or use CCW-assigned ID values to protect these sensitive data fields.
These sensitive fields were those which could be used to identify the prescriber (PRSCRBR_ID), the pharmacy, the plan (PLAN_CNTRCT_REC_ID) and benefit package (PLAN_PBP_REC_NUM),
or the plan’s formulary (FORMULARY_ID).  We are no longer required to encrypt the prescriber, pharmacy, plan or formulary identifiers.
3 For PDE files 2006-2013, the CCW_PHARM_ID  is used to identify the pharmacy. Beginning with 2014 PDE, the NCPDP_ID is used.
4 For PDE files 2006-2013,  the CCW_PRSCRBR_ID is used to identify the prescriber.  Beginning with 2014 PDE the actual prescriber identifier and the associated qualifier code
(PRSCRBR_ID and PRSCRBR_ID_QLFYR_CD variables) are used. The CCW Prescriber ID variable (and the associated PDE_PRSCRBR_ID_FRMT_CD) was retired after the 2013 data file.
5 Starting in 2010, these variables were moved from PDE to the Formulary file.
